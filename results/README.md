# Results Summary

This folder contains visualisations and communication network outputs generated during the analysis.

## Expression Analysis

- CES1 expression is heterogeneous across tumour and immune subsets.
- Tumour-associated macrophages showed notable CES1-associated patterns.

## Cell–Cell Communication (CellChat)

Global communication analysis identified:

- Enriched ligand–receptor interactions between tumour and immune cells.
- Differential pathway-level signalling activity.

Signalling pathways analysed include:
- MIF
- GALECTIN
- CCL
- CD40
- CD99
- LCK
- ICAM

Visual outputs:
- Circle plots
- Bubble plots
- Chord diagrams
- Pathway interaction networks

## Subcluster-Level Analysis

- Immune subclusters exhibited distinct signalling strengths.
- Communication heterogeneity was observed within macrophage populations.

## Ligand–Target Activity (NicheNet)

- Identified candidate upstream ligands driving downstream transcriptional changes.
- Highlighted tumour–immune regulatory interactions associated with CES1.

These results provide systems-level insight into CRC tumour microenvironment signalling architecture.
